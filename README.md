Description :
Bumaye App est une application Android de gestion des clients pour la maison de haute couture "Maison BUMAYE".
Fonctionnalités :

Ajout, modification et suppression de clients
Formulaire complet des mesures de couture
Liste avec RecyclerView et recherche en temps réel
Calcul automatique du reste à payer
Affichage des détails + photo du client

Bonus réalisés : Photo depuis la galerie + Recherche instantanée
Technologies : Kotlin + XML classique + View Binding + Données en mémoire
Design : Style Material clair avec CardView blanc, violet élégant et interface simple et professionnelle.
